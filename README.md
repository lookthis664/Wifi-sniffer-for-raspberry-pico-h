## **Features**
- [x] Have all the nearby Wi-Fi networks
- [X] Know if the Wi-Fi networks are public or private
- [X] Get the dBm of the Wi-Fi networks
- [X] Get the channels of the Wi-Fi networks

## **_Requirements_**
```
1-Raspberry pico h
2-ESP8266 wifi hat from sb components
```
